# Customer Retention & Churn Analysis Dashboard

## Project Overview
This project analyzes customer churn data to understand why customers leave and which factors affect customer retention.

## Tools Used
- Power BI
- CSV Dataset

## Dataset
Telco Customer Churn Dataset

## Analysis Done
- Total Customers
- Churned vs Retained Customers
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method

## Dashboard
The dashboard was created in Power BI using cards, donut chart, bar charts, and slicers.

## Key Insights
- Customers with month-to-month contracts show higher churn.
- Churn varies based on internet service type.
- Payment method has an impact on churn behavior.
- Retained customers are higher than churned customers.

## Recommendations
- Provide offers for month-to-month contract customers.
- Improve service quality for high-churn internet service users.
- Give loyalty benefits to long-term customers.
- Track churn regularly to improve customer retention.
